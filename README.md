# JobMaiKub

## 🇺🇸 English

JobMaiKub is a modern full-stack career development platform designed to help users discover their ideal career path and build the skills needed to achieve their goals. By combining AI-powered career recommendations, personalized learning roadmaps, and progress tracking, the platform provides a structured and engaging learning experience for students, graduates, and professionals.

Users can explore various career opportunities, receive AI-generated career matches based on their interests and skills, follow customized learning paths, and monitor their progress through achievements and analytics. The platform is built with modern web technologies, focusing on responsive design, scalability, and an intuitive user experience.

### ✨ Key Features

* 🤖 AI-powered career matching based on user profiles
* 💼 Explore careers and discover suitable career paths
* 📚 Personalized learning roadmaps and skill development
* 📈 Track learning progress, milestones, and achievements
* 📰 Career news and industry updates
* 📱 Fully responsive design for desktop and mobile devices
* ⚡ Fast, modern, and user-friendly interface

## 🇹🇭 ภาษาไทย

JobMaiKub เป็นแพลตฟอร์มพัฒนาอาชีพแบบ Full-Stack ที่ออกแบบมาเพื่อช่วยให้ผู้ใช้งานค้นพบเส้นทางอาชีพที่เหมาะสม พร้อมพัฒนาทักษะที่จำเป็นสำหรับการเติบโตในสายงานที่สนใจ ผ่านระบบแนะนำอาชีพด้วย AI (AI Career Matching) แผนการเรียนรู้เฉพาะบุคคล (Learning Path) และระบบติดตามความก้าวหน้า (Progress Tracking)

ผู้ใช้งานสามารถสำรวจอาชีพที่หลากหลาย รับคำแนะนำด้านอาชีพจาก AI ตามทักษะและความสนใจ วางแผนการเรียนรู้ผ่านเส้นทางที่ออกแบบเฉพาะบุคคล ติดตามความก้าวหน้าด้วยสถิติและความสำเร็จ รวมถึงติดตามข่าวสารและแนวโน้มของตลาดแรงงาน ทั้งหมดนี้ถูกพัฒนาด้วยเทคโนโลยีเว็บสมัยใหม่ โดยให้ความสำคัญกับประสิทธิภาพ ความยืดหยุ่นของระบบ และประสบการณ์การใช้งานที่ดีบนทุกอุปกรณ์

### ✨ คุณสมบัติเด่น

* 🤖 ระบบแนะนำอาชีพด้วย AI ตามทักษะและความสนใจของผู้ใช้
* 💼 สำรวจสายอาชีพและข้อมูลที่เกี่ยวข้อง
* 📚 เส้นทางการเรียนรู้ (Learning Path) เพื่อพัฒนาทักษะอย่างเป็นขั้นตอน
* 📈 ระบบติดตามความก้าวหน้า พร้อมสถิติและความสำเร็จ
* 📰 ข่าวสารและบทความด้านอาชีพ
* 📱 รองรับการใช้งานทั้งบนคอมพิวเตอร์และอุปกรณ์พกพา
* ⚡ อินเทอร์เฟซทันสมัย ใช้งานง่าย และตอบสนองรวดเร็ว
